About The Project

This project, "The Maze," was inspired by a personal interest in puzzle games and challenges. The goal was to create a game that not only entertains but also stimulates cognitive skills. I started working on this project in early 2024 as part of a personal initiative to dive deeper into game development. This is also a Portfolio Project for Holberton School, showcasing the skills learned throughout the program.
